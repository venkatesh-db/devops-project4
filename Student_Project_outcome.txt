Are there bugs or vulnerabilities?

# Layer4- Jenkins 

✅ Jenkins Pipeline – 8 Point Summary

Agent & Tools Setup
Uses Jenkins agent + configures SonarScanner and environment variables.

Code Checkout
Pulls latest code from GitHub (main branch).

Dependency Installation
Installs Python dependencies using pip3.

SonarQube Scan
Analyzes code for bugs, vulnerabilities, and code quality.

Quality Gate Validation
Stops pipeline ❌ if code does not meet quality standards.

Docker Image Build
Builds Docker image for the application.

Docker Push
Logs into DockerHub securely and pushes the image.

Kubernetes Deployment
Updates deployment, applies configs, and performs rolling update 🚀
